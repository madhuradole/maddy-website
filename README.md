# Madhura Dole Portfolio

Static portfolio site for `madhuradole-website.vercel.app`.

## Files

- `index.html` contains the page markup, styles, and content.
- `vercel.json` configures Vercel to serve the static HTML page.

## Preview locally

From this folder:

```sh
python3 -m http.server 4173
```

Then open:

```txt
http://localhost:4173
```

## Deploy

Deploy this folder to the existing Vercel project:

```sh
vercel --prod
```

If Vercel asks to link the project, choose the existing `madhuradole-website` project.
